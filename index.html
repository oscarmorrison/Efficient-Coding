Open Git in Web +

A simple script that allows you to simply type openGit in terminal (unix command line) and it will quickly open the web url address of the remote repository. This is very handy when you need to add extra collaboratos, want to check stats, or need to approve a pull request.

New Repository from CLI

A script that you to type newGit in terminal (unix command line) and it will create initialise the current directory as a git repository, add all files, commit with commit message "Initial Commit" and then create a new private repository on bitbucket and push code.
